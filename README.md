### What I learned

I learned about :

- How to make profile picture above another shape
- svg background

```css
.image-container {
  margin: 0;
  padding: 0;
  position: relative;
  height: 4rem;
}

.image-container > img {
  border-radius: 100%;
  border: 4px solid white;
  transform: translateY(-50%);
}
```
